extract to a folder where you want
in the folder there should be the:
-oldMansFolderComparator folder
-oldVersion folder
-newVersion folder
-readme.txt file // optional

how to use:
-in oldVersion put the folder of the previous version you want to compare
-in newVersion put the folder of the new version you want to compare
-in oldMansFolderComparator run the folderComparator.exe
-a file named changeLogs.txt will be created at the root folder
